# Vendor (Ventas) - Proyecto

Este repo agrupa el módulo **Vendor** del ecosistema ERP modular.

## Qué es Vendor

Vendor es el módulo de **ventas/comercial**. Aquí viven:
- Clientes (CRM básico)
- Órdenes con líneas y negociación (override de precio por línea)
- Pagos manuales por transferencia
- Visibilidad de envíos (tracking read-only desde Shipments)
